Running iPhone Applications as Widgets

One of the cool things about iPhone apps is that they're lightweight and easy to use – which makes them perfect candidates for widget conversion.

To convert an iPhone application into a widget, select Create Widget from Code from the main menu and enter the following:

<iphone src="http://www.some-iphone-app-url" />

Where src points to the URL of the iPhone application you want to convert into a widget. Press Create Widget and the iPhone application should appear on your screen just like any other widget. For best performance, install Apple's Safari 3.0 beta (some iPhone apps require it).

Note: This support is still experimental in 0.1a and will be finalized in a later version.


Setting Preferences in 0.1a

Although this build doesn't have a Preferences user interface, some application preference keys are already in place. If you have a copy of Apple's Property List Editor (included with Apple's Developer Tools) then you can modify these keys by opening:

~/Library/Preferences/com.mesadynamics.AmnestyHypercube

Alternatively, open your Terminal application and use the defaults command to modify an individual preference key. For example, typing the following into the Terminal:

defaults write com.mesadynamics.AmnestyHypercube PrefUISound NO

will turn off the user interface sound effects in the Hypercube.


This table describes the preference keys supported by Amnesty Hypercube 0.1a:

PrefLaunchHidden If YES, always launch with desktop widgets hidden.
PrefNoClickImport If YES, activate the NoClick widget creation technology so that it functions system-wide. This feature, which is normally activated only when the Provider Gallery is visible, automatically creates a widget when web widget HTML code is copied to the clipboard.
PrefReduceCPU If YES, set Amnesty Hypercube's process scheduling to "nice". This helps keep CPU usage down if multiple widgets (esp. Flash-based widgets) are open at the same time.
PrefUISound If YES, user interface sound effects in the Hypercube are turned on.
PrefCreateSound If YES, the sound effect used by NoClick when it creates a widget is turned on.
