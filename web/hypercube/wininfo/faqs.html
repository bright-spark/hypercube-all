Questions

1. Why do some widgets appear in the Widget Gallery with a default Gear icon while some have custom icons?

2. Why does the Hypercube seemingly close by itself sometimes?

Answers

1. Why do some widgets appear in the Widget Gallery with a default Gear icon while some have custom icons?

If Amnesty Hypercube cannot download an icon from a widget provider, the default Gear icon will be used. In a future version of Amnesty Hypercube you'll be able to customize your own widget icons.

2. Why does the Hypercube seemingly close by itself sometimes?

If you click on an external web page link inside a widget, the link will be opened in your default web browser. Because your web browser cannot be displayed while the Hypercube is opened, the Hypercube closes when an external link is clicked. We are currently working on a way to minimize this context shift in a future version of Amnesty Hypercube.
