The following issues are confirmed to exist in the latest release of Amnesty Hypercube. If you have an issue that is not listed here, please let us know.

Resolution Changes Not Tracked

Amnesty Hypercube ignores changes to screen resolution that occur while the application is running. This means that widgets will not appear in their correct locations and may move offscreen. In addition, the Hypercube may no longer cover your entire screen real estate. There is no current workaround to this issue except to avoid resolution switches while Amnesty Hypercube is running.
