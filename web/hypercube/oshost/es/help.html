Ayuda

Amnesty™ Hypercube es una aplicación de OpenSocial que le permitirá reunir y compartir Web Widgets con sus amigos. ¿Qué es un Web Widget? Un web widget es una mini-aplicación la cual está incorporada dentro de un blog o de una página web. Ejemplos de esto son pases de diapositivas de fotos, reproductores de audio de red, juegos flash o video incrustado. Por ejemplo, un video de YouTube podría ser considerado como widget. Como también podría serlo un juego flash procedente de Miniclip. La cosa importante que los web widgets tienen en común es que son publicados como pequeños fragmentos de código HTML.

Por medio de Hypercube podrá convertir a estos fragmentos de código en widgets que se podrán ejecutar dentro de OpenSocial.

El truco está en cómo hallar estos fragmentos de código y ahí es donde la Galería de Proveedores de Hypercube puede ayudar. La Galería de Proveedores es una lista de sitios web que publican widgets en forma de fragmentos de código. Nosotros mantenemos la Galería de Proveedores actualizada con los widgets nuevos que vayan saliendo, por lo que este es un buen lugar para consultar con frecuencia. Cada sitio tiene un procedimiento diferente para obtener el código, pero una vez que se ha obtenido este sólo hay que cortar y pegar el fragmento de código dentro de Hypercube para comenzar a usarlo directamente.

Otro lugar conveniente para encontrar Widgets es la pestaña de Widgets principales de Hypercube. Nosotros mantenemos actualizada esta pestaña con los mejores widgets que encontramos para compartir con usted.

Tenemos muchísimos planes para Hypercube en OpenSocial, incluyendo la integración con nuestra plataforma gratuita de widgets, Amnesty Hypercube. De este modo, podrá acceder a sus widgets directamente en su escritorio, en la barra lateral de Vista, del Dashboard de Apple o aquí en OpenSocial. Al continuar perfeccionando ambos productos, nos aseguraremos de mantenerle informado.
