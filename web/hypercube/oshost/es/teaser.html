¡Instale Hypercube hoy!

Amnesty™ Hypercube es una aplicación de OpenSocial que le permitirá reunir y compartir Web Widgets con sus amigos. ¿Qué es un Web Widget? Un web widget es una mini-aplicación la cual está incorporada dentro de un blog o de una página web. Ejemplos de esto son pases de diapositivas de fotos, reproductores de audio de red, juegos flash o video incrustado.

Si no puede encontrarnos en el directorio de esta aplicación de red y ésta le permite añadir manualmente una URL, utilice la siguiente URL para instalar Hypercube:

    http://www.amnestywidgets.com/hypercube/oshost/hypercube.xml
