Install Hypercube Today!

Amnesty™ Hypercube is an OpenSocial application for MySpace that allows you to collect and share web widgets with your friends. What's a web widget? A web widget is a mini-application that is supposed to be embedded inside a blog or home page. Examples include photo slideshows, streaming audio players, flash games and embeddable video.

If you've already installed Hypercube, you need to login before using it.

Otherwise, you can install Hypercube and login from Hypercube's application profile page.
